# Career Accelerator

An automated job application platform that helps you move up the career ladder faster.

## Features

- Automated job applications on LinkedIn and Indeed
- Customized job search based on salary, work mode, location, and role
- Resume and cover letter optimization for each application
- Email automation for applications requiring email submission
- Form filling automation
- Application tracking dashboard

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the development server with `npm start`
4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser

## Technologies Used

- React
- Material UI
- Firebase Authentication
- Formik & Yup for form validation
- Axios for API requests
