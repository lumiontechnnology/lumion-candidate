Backend for Lumion Candidate

Run locally:

1. Install: `npm install` (from the `server` folder)
2. Dev: `npm run dev`
3. API base: `http://localhost:4000/api`

Routes:
- `GET /health` health check
- `GET /api/stats` application statistics
- `GET /api/jobs/saved` saved jobs
- `GET /api/jobs/recommended` recommended vacancies
- `GET /api/notifications` notifications
